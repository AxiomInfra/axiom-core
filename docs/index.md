# Axiom Core SDK (v1.0)

Axiom Core is a local semantic transformation SDK that turns raw input into
non-identifying structured context for downstream reasoning. It is designed
to keep raw data on-device while preserving useful structure for models.

## What this docs site covers

- User Guide for integration and configuration
- Developer Guide for SDK internals and testing
- API Reference
- Architecture and Security details
- Integration guides and examples
